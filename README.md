# yup-es

ES-es locale for yup

## Installation
Choose your package manager:

```bash
yarn add yup-es | npm i yup-es
```

## Usage

### JS
```javascript
const yup = require('yup')
const es = require('yup-es')

yup.setLocale(es)
```

### ES6 / Typescript
```typescript
import * as yup from 'yup'
import es from 'yup-es'

yup.setLocale(es)
```
